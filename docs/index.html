Spring Boot · Java 17 · H2/JPA

CardLifecycleApi

A demo API that issues masked cards, manages lifecycle state, and authorizes spending within a persisted daily limit.

Masked responses

Full PAN values never leave card API responses.

Daily authorization

POST /authorize declines blocked cards and amounts above the remaining limit.

Persistent demo state

H2/JPA preserves card and daily spending state across restarts.
